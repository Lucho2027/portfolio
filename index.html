ProdTrack App

This app is intended for front line leaders, Industrial/Manufactring Engineers and supervisors on a manufacturing facility setting. This Full-Stack app was created using ReactJS, HTML, CSS, Express and PostgreSQL created to provide a tracking tool for production efficiency. It allows the user to record production hour by hour results. This results are compared live with the hourly goal this app calculates against the output desired a work center is performing. In addition, once a shift is over this form is summarized and saved on a database. This allows the engineers to acces the information to understand and look for trends on productivity.
